Hexo主题选择及配置

Hexo博客主题选择

不同的会使你的博客主题有不同的风格，例如博客的主页、标题以及图标等。选择一个自己喜欢的主题，配置自己个性化的博客。如果你懂一些编程，可以自己动手折腾，为自己的博客写一款主题或者在其他主题的基础上做一些修改。
不过除了自己写你也可以在Hexo的官网上找一款自己喜欢的主题，Hexo官网为我们提供了数十款精美的主题。 戳→ https://hexo.io/themes/ 。
下面我们以一款十分流行的主题NexT为例，教大家如何来配置主题。

主题安装

NexT为我们提供了详细的使用文档。

主题下载

在你的blog目录下右键运行 git bush here，然后执行一下命令：

git clone https://github.com/iissnan/hexo-theme-next themes/next

这样你便可以从github上将next主题克隆到你的博客的主题下。克隆完成后如下：

同时，你在博客下的theme文件夹下可以看到一个next文件夹。

启用NexT文件

将NexT主题克隆完成后，打开站点配置文件，找到theme字段，并将其值改为next。

这里需要注意区分两个配置文件：

站点配置文件：是你的hexo博客目录下的 _config.yml 文件。
主题配置文件：是themes/next目录下的 _config.yml 文件。

博客的基本配置

语言设置

每个主题都会配置几种界面显示语言，修改语言只要编辑 站点配置文件，将 language 设置成你所需要的语言。
例如选用简体中文，则配置为：

language: zh-Hans

关于主题具体支持哪些语言，可以到 themes/next/languages 目录下查看。

网站标题及作者

打开站点配置文件，修改下面这些值：

title: Hexo #博客标题
subtitle: #博客副标题
description: #博客描述
author: #博客描述

冒号后面必须有空格，否则会编译错误。

域名及文章连接

打开站点配置文件，修改这些值：

url: http:// #你的博客地址
root: /#博客根目录，如果你的博客的网址在二级目录下，在这里填上
permalink: /blog/:title/html ##:year/:month/:day/:title/ 原来跟着年月日，建议缩短为一个 blog目录加标题，以后取文章标题的时候不要重复就可以。
permalink_defaults: #可不填

NexT主题Scheme设置

NexTNexT 主题目前提供了3中风格类似，但是又有点不同的主题风格，可以通过修改 主题配置文件 中的 Scheme 值来启用其中一种风格，如选择 Pisces 分栏风格，只要把另外两个用#注释掉即可。

# Schemes
# scheme: Muse
# scheme: Mist
scheme: Pisces
